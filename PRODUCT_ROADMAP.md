# OpenClaw GEO 产品迭代路线图 (Product Roadmap)

> 当前版本：v2.0 (Stable) - 全平台全自动监测版
> 更新日期：2026-02-05

本文档规划了 OpenClaw GEO 系统从当前稳定版向企业级智能平台演进的迭代路径。

---

## 🚀 v2.x 系列：数据与效能深化 (Current Focus)

### v2.1 - 数据可视化与导出 (Next Step)
**目标**：解决目前仅有文本报告、无法做二次分析的痛点。
- [ ] **多格式导出**：支持生成 Excel (`.xlsx`) 和 CSV 格式报告，包含原始问答数据。
- [ ] **HTML 仪表盘**：生成单页 HTML 报告，包含基础的饼图/柱状图（竞品提及占比）。
- [ ] **结构化存储**：从散乱的 JSON 文件升级为 SQLite 或按日期归档的结构化目录。

### v2.2 - 性能与并发优化
**目标**：解决串行监测速度慢的问题，提升执行效率。
- [ ] **并发执行**：引入 `asyncio` 或线程池，实现 Deepseek、Kimi、Doubao 同时提问，速度提升 4 倍。
- [ ] **流式输出**：在终端实时展示大模型的生成过程（打字机效果），减少用户等待焦虑。
- [ ] **断点续传**：如果任务中途意外中断，下次启动可选择继续未完成的任务。

### v2.3 - 深度洞察引擎 (Deep Insight)
**目标**：从“看到数据”升级为“看懂差距”。
- [ ] **差距分析 (Gap Analysis)**：自动对比“竞品被提及的优点” vs “联想缺失的点”。
- [ ] **关键词雷达**：提取竞品高频出现的“褒义词”和“场景词”，生成 SEO/GEO 关键词建议库。
- [ ] **情感分析**：判断提及内容是正面、负面还是中性（目前仅统计了“是否提及”）。

---

## 🛠 v3.0 系列：平台化与自动化 (Long Term)

### v3.0 - Web 可视化工作台
**目标**：脱离黑底白字的命令行，提供现代化的 Web 操作界面。
- [ ] **Web UI**：基于 Streamlit 或 React 构建管理后台。
- [ ] **趋势看板**：展示“提及率趋势图”（周/月变化曲线）。
- [ ] **配置管理**：在网页上直接修改 Prompt、添加新意图、管理 API Key。

### v3.1 - 自动化闭环
**目标**：实现“无人值守”的长期监测。
- [ ] **定时任务**：内置 Crontab 调度，每天凌晨自动运行。
- [ ] **主动推送**：监测完成后，自动将简报推送到 飞书/钉钉/邮件。
- [ ] **预警系统**：当提及率低于 X% 或竞品声量暴涨时，触发紧急告警。

| :--- | :--- | :--- | :--- |
| **v2.1** | 1周 | Excel 导出, HTML 简报 | 需要做汇报 PPT 的运营人员 |
